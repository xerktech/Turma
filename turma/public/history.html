Daily figures are parsed from Claude transcripts (UTC days) and aggregated by each host agent across all of its worktrees — API-equivalent cost estimates that persist regardless of active sessions (killed, deleted, and pruned work still counts). By repo unifies a repo's usage across every host it runs on (matched by git origin); By host totals each host. Click a series in the legend to toggle it (click the group name to toggle all); hover a day for the breakdown. The table view carries the same numbers for the visible series.
