Cost figures are API-equivalent estimates computed from transcript token counts — sessions run on the shared subscription, so this is relative load, not a bill. Each host multiplexes worktree-backed sessions. Kill stops a session and removes it from the hub (worktree deleted) in one step, but keeps its git branch and usage history — the work stays in the repo, it just leaves the dashboard. Delete (on an already-stopped session) additionally erases the branch and history. "Restart (clear context)" relaunches a session with a fresh transcript and a new claude.ai/code name.
